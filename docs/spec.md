Intelligent Tool for Effective Assessment of Student Project Documentations

Department of Information Systems

### Assignment

Objectives which should be achieved:

1. Study and briefly describe machine learning and artificial intelligence techniques used for understanding and scoring the quality of text and images. Focus on Transformer-type models and large language models. Explore and compare different types of these models (e.g., GPT, LLaMA, BERT, etc.) and related services (e.g., OpenAI API).
2. Map existing approaches and tools for evaluating the quality of technical texts based on certain criteria (specified by the user or otherwise in advance).
3. Design the architecture of a tool that uses an appropriate machine learning model (or a combination of models) to assess the documentation of projects submitted in the courses _Formal Languages and Compilers_ and _Principles of Programming Languages_ at FIT. The tool's input will be a documentation in PDF or Markdown format and evaluation criteria provided in natural language. The output will (primarily) be consistent recommendations for point-based scoring according to each criterion.
4. Implement the proposed tool. For training the machine learning models, appropriately modify and utilize the dataset provided by the supervisor.
5. Evaluate the recommendations provided by the tool. Focus on the correctness of the results and the objectivity of the proposed point scores.
6. Discuss the benefits of the solution in terms of recommendation quality, usage cost, runtime, and time savings compared to the current fully manual project evaluation process.

#### Requirements for granting of credit for semester work

Points 1, 2 a 3.
